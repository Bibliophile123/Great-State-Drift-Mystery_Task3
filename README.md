
# 📋 Great Drift Mystery

A beginner-friendly Flutter application that demonstrates a horizontal task list with interactive task selection and deletion functionality. This project focuses on Flutter UI design, state management using `setState()`, and dynamic list updates.

---

## ✨ Features

- 📌 Horizontal scrollable task cards
- ✅ Select and deselect tasks
- 🗑 Delete the first task dynamically
- 🎨 Clean and modern Material Design UI
- 📱 Responsive layout
- ⚡ Beginner-friendly Flutter code

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Bibliophile123/Great-State-Drift-Mystery_Task3.git
```

2. Navigate to the project folder

```bash
cd Great-State-Drift-Mystery_Task3
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
